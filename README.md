# ldk_node_cli

This is a node based on LDK's sample node implementation. The CLI is intended to be used with the node and function just as `lncli` works for `LND`.

## Installation

``` bash
git clone git@github.com:Jennycj/ldk_node_cli.git
```

## Usage

``` bash
cd ldk_node_cli
cargo run --bin cli
```
